# AnasOS PWA

تطبيق ويب بسيط يعمل كتطبيق على الآيفون عبر Safari > Share > Add to Home Screen.

## Features
- Daily Log
- Quick Notes
- History
- Export JSON
- Export CSV
- Local storage on device
- Offline cache after first load

## Deploy
ارفع الملفات على GitHub Pages أو Netlify، ثم افتح الرابط من Safari على الآيفون وأضفه للشاشة الرئيسية.
