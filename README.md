# AnasOS V2

Personal daily log, memory recall tracker, quick notes, history, and export.

## Deploy on GitHub Pages

1. Upload all files to the root of your repository.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /(root).
6. Open: https://YOUR_USERNAME.github.io/REPO_NAME/

## Install on iPhone

Open the site in Safari > Share > Add to Home Screen.

## Data

Data is stored locally on the same device/browser using localStorage.
Use Export JSON regularly as a backup.
